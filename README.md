# 🐾 PetVet – Veterinary Clinic Management System

## 📌 Overview
**PetVet** is a desktop-based veterinary clinic management system developed using **Java** and **JavaFX**. The application provides a centralized platform for managing veterinary clinic operations, including pet records, owner profiles, appointment scheduling, and medical histories.

The system follows the **Model–View–Controller (MVC)** architecture to ensure clear separation between the application's data, user interface, and business logic. This design improves maintainability, scalability, and code organization.

PetVet aims to enhance the efficiency of veterinary clinics while providing pet owners with a convenient platform to manage their pets’ healthcare information.

---

## 🎯 Key Features

- Secure **user authentication** and registration system  
- **Pet profile management** for storing pet details and medical history  
- **Veterinary appointment scheduling**  
- **Medical record management** for diagnoses and treatments  
- **Role-based dashboards** for different users  
- Clean and modern **JavaFX UI with blue-themed design**

---

## 👥 User Roles

### 👤 Guest
Guests can explore the system before creating an account.

**Capabilities**
- View general clinic information
- Browse available veterinarians
- View basic service details
- Access registration and login pages

---

### 🧑‍💼 Pet Owner
Pet owners can manage their pets and appointments.

**Capabilities**
- Create and manage personal accounts
- Add and manage multiple pet profiles
- View pet information including breed, age, and medical history
- Track vaccination and treatment records
- View available veterinarians
- Book and manage veterinary appointments
- View appointment history

---

### 👨‍⚕️ Veterinarian (Doctor)
Veterinarians manage consultations and pet medical records.

**Capabilities**
- View scheduled appointments
- Access pet profiles and medical histories
- Record diagnoses and treatments
- Add prescriptions and medical notes
- Update appointment statuses

---

### 🛠 Admin
The administrator manages the entire system.

**Capabilities**
- Manage user accounts (pet owners and veterinarians)
- Monitor clinic operations
- Manage appointment schedules
- Maintain system data integrity

---

## ⚙ Core Modules

### 1️⃣ Authentication Module
- Login and registration system
- Secure authentication for all user roles
- Input validation and error handling

### 2️⃣ Pet Management
- Add, update, view, and delete pet profiles
- Store information such as:
  - Pet name
  - Species
  - Breed
  - Age
  - Owner details
  - Medical notes

### 3️⃣ Appointment Management
- Schedule veterinary appointments
- View appointment schedules
- Update appointment status:
  - Pending
  - Confirmed
  - Completed

### 4️⃣ Medical Record Management
- Record diagnoses and treatments
- Maintain treatment history for each pet
- Store prescriptions and medical notes

### 5️⃣ Dashboard System
Dedicated dashboards for each user role:

- **Pet Owner Dashboard** – Pet overview and appointment management  
- **Veterinarian Dashboard** – Appointment schedules and medical records  
- **Admin Dashboard** – System monitoring and management
